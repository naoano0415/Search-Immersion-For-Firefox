# Search Immersion For Firefox

**Google検索画面や新しいタブを、Google Nest Hub風のスマートディスプレイに変えるアドオンです。**

現状、リリーステスト版（ベータ版）ですので、一部環境で挙動が異なる場合があります。ご了承ください。

---
## 1.インストール方法
1. [Release](https://github.com/naoano0415/Search-Immersion-For-Firefox/releases)から最新版をダウンロードします。
2. ポップアップが出てくるので、「インストールを続行」をクリック
3. 「追加」をクリック

**2. でポップアップが出てこない場合**
1. [about:addons](about:addons)にアクセス
2. 右上の設定アイコンから、「ファイルからアドオンをインストール」を選択
3. [Release](https://github.com/naoano0415/Search-Immersion-For-Firefox/releases)からダウンロードしたファイルを選択

### 2. 使い方

1. Googleのトップページにアクセスします。
2. 自動的にSearch Immersionのインターフェースが表示されます。
3. 画面中央下のドックにある **[⚙️] (設定)** ボタンから、好みのデザインや連携機能を設定してください。
